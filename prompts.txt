Version v1.3 del sistema de parqueo publico.

Objetivo:
Construir un proyecto academico en Java usando NetBeans + Apache Ant, con interfaz grafica Swing, arquitectura por capas y persistencia en archivos .txt.

Alcance implementado:
- Registro de ingreso de vehiculos.
- Registro de salida de vehiculos.
- Registro manual de minutos de permanencia para la salida.
- Calculo del monto a pagar.
- Historial de vehiculos atendidos.
- Eliminacion de registros historicos.
- Interfaz grafica mejor organizada.
- Persistencia en archivo de texto.

Restricciones respetadas:
- No se usa consola.
- No se usa JOptionPane.
- Toda la interaccion ocurre dentro de ventanas Swing.

Refinamiento de la iteracion:
- Se mantiene la arquitectura por capas.
- La salida ya no depende de la hora real del sistema para calcular el cobro.
- El usuario indica manualmente los minutos de permanencia.
- El sistema calcula el monto con tarifa de `CRC 500` por hora o fraccion.
- El tiempo de permanencia queda guardado en el historial.
